### Trip Cost Calculator
```

flight = $450

days = 5

meals = $50

```

### Fuel Efficiency
```
# Gas Mileage
miles_driven = 350 miles
gallons_used = 12 gallons
mpg = miles_driven / gallons_used

// Convert to metric
km_driven = miles_driven in km
liters_used = gallons_used in liters
kpl = km_driven / liters_used
```

## Tips & Tricks

1. **Headers** - Organize your calculations
```
# My Budget
# Travel Expenses
# Total Cost
```

2. **Comments** - Add notes
```
// This is the price before tax
// Remember to update this monthly
```

3. **Reference Previous** - Use `prev`
```
100 + 50
prev * 2
prev - 25
```

4. **Chain Conversions**
```
20 inches in cm + 5 cm
100 USD in EUR - 10 EUR
```

5. **Complex Expressions**
```
(100 + 50) * 0.8 - 20
sqrt(16) + 2^3
pi * 5^2
```
